Responsive Multi-Page Dealership Website

Overview

This project is a fully responsive dealership website designed for Sam Allen Ford. It features a clean layout, modern UI components, and mobile-first design principles. The site includes inventory sections, service information, and contact details.

Features

  • Multi-page responsive layout
  • Custom CSS components and grid system
  • Interactive navigation and image sections
  • Optimized for mobile, tablet, and desktop

Tech Stack

HTML5, CSS3, PHP, and MySQL
