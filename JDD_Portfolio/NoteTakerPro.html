Android/Kotlin Productivity App

Overview

Note Taker Pro is a modern Android note-taking application built using Kotlin and Jetpack Compose. It provides a clean, intuitive interface for creating, editing, and organizing notes with persistent local storage.

Features

  • Create, edit, and delete notes
  • Local JSON file storage for persistence
  • Jetpack Compose UI with smooth animations
  • Responsive layout for phones and tablets

Tech Stack

Kotlin, Jetpack Compose, Android Studio, MVVM Architecture
